# Jarvis
Safe AI assistant

# Python Version
To run Jarvis you need Python 3.10-3.12

# Commands for runnings
Commands to run english client on Windows:
1) cd (Where it's jarvis)\Jarvis\clients\Windows\Eng
2) pip install vosk sounddevice requests pygame python-Levenshtein colorama edge_tts
3) python jarvis_wake.py

Commands to run english client on Linux:
1) cd (Where it's jarvis)\Jarvis\clients\Linux\Eng
2) source jarvis-env/activate/bin
3) pip install vosk sounddevice requests pygame python-Levenshtein colorama edge_tts
4) python jarvis_wake.py
